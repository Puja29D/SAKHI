# Sakhi - Safe Digital Support Platform

Sakhi is a trauma-informed web platform designed to support survivors of gender-based violence.

## Prerequisites

- Python 3.8+
- Pip (Python Package Manager)

## Setup & Run

1.  **Create a Virtual Environment** (Recommended):
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Application**:
    ```bash
    python app.py
    ```

4.  **Access the Platform**:
    Open your browser and navigate to: `http://127.0.0.1:5000`

## Features

-   **Quick Exit**: A "Quick Exit" button is available on every page to immediately redirect to Google.
-   **Survivor Dashboard**: Mood tracking, daily affirmations, and access to resources.
-   **Rights & Awareness**: Tabbed information about legal rights and reporting options.
-   **Healing Tools**: Breathing exercises, grounding techniques, and a digital doodle canvas.
-   **Parent/Guardian Guide**: Resources and advice for parents supporting their children.

## Safety Note

This platform uses local storage for some interactive elements and does not persistently store sensitive personal narratives to protect user privacy.
